# Backend Services

This directory contains the microservices that power the platform.

## Services
- **audit-service**: centralized audit logging.
- **policy-service**: OPA wrapper for authorization decisioning.
- **tenant-service**: multi-tenancy management.
- **ingestion-service**: document processing pipeline.
- **rag-service**: retrieval augmented generation.
- **llm-gateway**: centralized point for LLM interaction.
- **mcp-gateway**: Model Context Protocol integration.
- **workflow-service**: Temporal workflow definitions and workers.
